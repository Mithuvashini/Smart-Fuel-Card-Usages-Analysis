# Smart Fuel Card Performance Dashboard

## Overview
This project analyzes **smart fuel card transactions** to monitor fuel consumption, spending patterns, and operational efficiency across states, districts, cards, and fuel stations.  
The dashboard is built to help management **track cost leakage, usage trends, and high-impact fuel stations** quickly.

## Key Objectives
- Track total fuel spend and quantity consumed
- Compare diesel vs petrol usage
- Identify top-performing fuel cards and fuel stations
- Monitor monthly trends in fuel quantity and cost
- Analyze transaction mode usage (Card vs OTP)

## Dashboard KPIs
- **States Covered:** 3  
- **Districts Covered:** 32  
- **Fuel Stations:** 95  
- **Cards Issued:** 34  
- **Total Transactions:** 1,125  
- **Total Amount:** ₹95,20,430  
- **Total Quantity:** 1,02,080 Litres  
- **Average Cost per Litre:** ₹93  

## Visualizations Included
- **Quantity (Litres) by Product Type**  
  - Diesel vs Petrol distribution  
- **Monthly Fuel Quantity Trend**  
  - Identifies peak and low usage months  
- **Total Amount by Fuel Card**  
  - Highlights high-spend cards  
- **Monthly Spend Trend**  
  - Detects spikes and anomalies  
- **Transaction Mode Analysis**  
  - Card vs OTP usage comparison  
- **Fuel Station-wise Quantity Consumption**  
  - Identifies top-consuming stations  

## Filters & Slicers
- Transaction Mode (Card / OTP)
- Name of Card
- State (Andhra Pradesh, Karnataka, Tamil Nadu)

## Tools & Technologies
- **Power BI**
- **DAX** for calculated measures
- **Excel** as the data source

## Business Insights
- Diesel accounts for **~98%** of total fuel consumption  
- Majority of transactions occur via **Card (≈84%)**, not OTP  
- Fuel usage and spending peak around **August–July**  
- A small number of fuel stations contribute to a **large share of total volume**

## Use Case
This dashboard is suitable for:
- Fleet & logistics teams
- Finance & cost-control teams
- Operations managers monitoring fuel efficiency
- Auditing fuel card usage

## Notes
- Data shown is for analytical and demonstration purposes.
- Dashboard design prioritizes **clarity, trend visibility, and decision-making speed**.
